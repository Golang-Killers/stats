# stats

Calculate stats over newline separated numbers on stdin.

# Usage

```
$ cat results.txt
982
651
824
850
699
734
782
1072
728
809
1022
780
865
1061
994
881
899
804
736
838

$ cat results.txt | stats
min 651.0000
max 1072.0000
sum 17011.0000
mean 850.5500
median 850.5500
modes []
stdev 122.0459
```
